# snakepy

**A snake game made by Pygame and Python!**

Choose speed on the menu - 6 speeds (New speed: Superior)

Use arrow key or WASD to control snake

You also can import custom speed file

**Latest version:1.2**

---

*If you run the source code, make sure to have Python and Pygame (type pip install pygame in CMD if you have Python)*
